(static) eachPlanet(planets, callback, opts)

Loop through each given planet while handling getting buildings, unhappiness and errors.

Parameters:
Name Type Description
planets array

the list of planets you want to loop through
callback function

a thenable to run on each planet
opts object

options (set force to true to ignore unhappiness checks here)
Source:
  • lacuna/modules/empire.js, line 172
Examples
// Go thru each colony
lacuna.empire.colonies.then((colonies) => {
  return lacuna.empire.eachPlanet(colonies, (colony, buildings) => {
    // Do stuff with colony or buildings.
    // Erorrs thrown in here are logged to the console and don't kill the current task.
  })
})
// Go thru each colony ignoring unhappiness checks (NOT RECOMMENDED)
lacuna.empire.colonies.then((colonies) => {
  return lacuna.empire.eachPlanet(colonies, (colony, buildings) => {
    // Do stuff with colony or buildings.
    // Erorrs thrown in here are logged to the console and don't kill the current task.
  }, {force: true})
})

(static) findPlanets(planet, skip) → {array}

This function is for handling the --planet argument.

It can be one of three things:

  • the string 'all'
  • a planet name
  • an array of planet names
Parameters:
Name Type Description
planet string | array

the planet names you want to find
skip string | array

the planet names you want to skip
